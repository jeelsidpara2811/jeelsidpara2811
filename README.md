## About Me

I'm Jeel, a data scientist and AI/ML engineer who likes projects that go all the way from raw pixels or raw numbers to something a user can actually run. That's shown up as an OCR pipeline that reads stamp numbers off steel billets on a factory floor, a CLIP-based visual search tool for finding look-alike shoes, a clustering model that segments customers by behavior, and a FastAPI service that serves live stock data in a Docker container. I also like understanding the hardware underneath the software, which is why I built an Arduino Li-Fi link that sends text over an LED bulb and reads it back with a solar panel.

- 🔭 Currently building applied computer-vision and ML projects, end to end (model → API → deployment)
- 🌱 Deepening my skills in deep learning, model deployment, and data engineering
- 💬 Ask me about OCR/computer vision, clustering & segmentation, or building small ML-backed APIs
- 📫 Reach me at **jeelsidpara2811@gmail.com** or on [LinkedIn](https://www.linkedin.com/in/jeel-sidpara-276208359/)
